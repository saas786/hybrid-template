# Hybrid\\Template

Hybrid Template is a set of helper functions for setting the location, locating, and filtering templates.

## Requirements

* WordPress 4.9+.
* PHP 5.6+ (preferably 7+).
* [Composer](https://getcomposer.org/) for managing PHP dependencies.
